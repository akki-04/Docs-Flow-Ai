# DocFlow AI

## Problem Statement

Companies manually analyze documents like resumes, invoices, agreements, and reports.
This process is slow and error-prone.

## Solution

DocFlow AI allows users to upload documents, extract structured information using AI, and view the results through a dashboard.

## Target Users

- HR
- Recruiters
- Finance Teams
- Students

## Version 1 Features

- User Registration
- Login
- JWT Authentication
- Upload PDF
- Analyze Document
- View Result
- Dashboard
